Class color

Known Subclasses:
  • vector

slum datatype to hold color data. The "internal" variable is a list that defines all the possible keys for this datatype. As default, they are [r,g,b], but they can be overriden by derivated classes, like vector, which defines it as x,y,z. This datatype is defined to simplify a shader implementation, specially when defining slum shader parameters.
